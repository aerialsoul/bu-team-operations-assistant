## HOW THIS FILE WORKS

This file tracks two types of scheduling activity:

1. INITIAL SCHEDULING (early season)
   Games exist in GotSport but have no confirmed time 
   or field yet. Manager is working to get all games 
   scheduled. This is the highest stress period of the 
   season - use this tracker to stay on top of every 
   open item.

2. RESCHEDULING (mid/late season)
   Games already scheduled that need to move for any 
   reason. Tracked separately from initial scheduling.

### Status Values Used in This File:
- 🔲 NOT YET CONTACTED
- 📨 CONTACTED - AWAITING RESPONSE
- 🔄 AGREED - PENDING SYSTEM UPDATE
- ✅ CONFIRMED IN GOTSPORT
- ❌ DECLINED - EXPLORING ALTERNATIVES
- 🔁 RESOLVED - NO CHANGE NEEDED

### Follow-Up Rule:
If any item has been in 📨 CONTACTED - AWAITING RESPONSE 
status for 5+ days, the assistant will flag it:
"📌 Follow-up needed: [Opponent] hasn't responded 
since [date]. Consider sending a follow-up message."
